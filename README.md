# Pong
![Pong](/img/Pong.png?raw=true)

## Controls
| Key                 | Description            |
| ------------------- | ---------------------- |
| <kbd>space</kbd>    | Play                   |
| <kbd>W</kbd>        | Move first player up   |
| <kbd>S</kbd>        | Move first player down |
| <kbd>UP</kbd>       | Move second player up  |
| <kbd>DOWN</kbd>     | Move second player down|

## Quick Start

Dependencies: [SDL2], [SDL2_ttf]
```console
$ cd Pong/src/
$ chmod +x build.sh
$ ./build.sh
```
